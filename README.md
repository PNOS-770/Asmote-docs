# Asmote

**Asmote** 是一款跨平台终端模拟器与远程管理工具，基于 Flutter 构建。支持 SSH、本地 Shell、串口、Telnet 等多种连接协议，集成分屏终端、文件传输、脚本自动化、端口转发等功能，适用于开发者和运维人员的日常服务器管理。

## 核心特性

### 连接管理

- **多协议支持**：SSH（密码/密钥/Agent 认证）、本地终端（PowerShell、CMD、WSL、bash）、串口、Telnet
- **跳板机与代理**：SSH ProxyJump、SOCKS5 代理
- **会话树**：分组管理主机，支持搜索、筛选、排序、固定常用主机
- **快速连接**：一键发起临时连接
- **自动重连**：断线后自动尝试恢复
- **连接池**：复用 SSH 连接，减少重复握手

### 分屏终端

- **多布局**：水平分割、垂直分割、网格布局，支持保存/加载布局模板
- **面板管理**：切换、最大化/还原、关闭面板
- **广播输入**：同时向所有面板发送键盘输入
- **终端搜索**：支持正则表达式搜索终端输出
- **块选择**：行选择与块选择模式一键切换
- **外观定制**：字体/字号/行高、配色方案、背景图片（支持每面板独立配置）
- **滚动缓冲**：可配置的回滚行数

### 文件传输与浏览

- **SFTP 传输**：基于 Rust 原生加速引擎，支持上传/下载/批量队列
- **传输控制**：暂停、恢复、取消、自动重试（可配置重试次数与延迟）
- **拖拽上传**：支持拖拽文件到终端或文件树
- **远程文件管理**：浏览远程目录树，新建/重命名/删除文件与文件夹
- **文件查看器**：在线预览远程文件内容
- **文件编辑器**：直接编辑远程文件并保存

### 脚本自动化

- **脚本管理**：创建/编辑/删除脚本，支持文件夹分组
- **变量与环境变量**：脚本参数化与上下文配置
- **预检命令**：执行前检查前置条件
- **工作流**：将多个脚本编排为顺序执行序列
- **批量模板**：预配置目标主机与输出类型，一键批量执行
- **定时调度**：Cron 表达式设置定时执行，支持时区偏移
- **触发器**：会话连接成功或提交特定命令时自动执行脚本（支持匹配/正则）
- **快捷键绑定**：为脚本绑定键盘快捷键
- **失败策略**：失败时可选择跳过、停止或重试

### 端口转发

- **本地转发**：将本地端口转发到远程目标
- **远程转发**：将远程端口转发到本地
- **SOCKS 代理**：动态端口转发
- **自动启动**：连接时自动启用指定转发规则
- **模板管理**：将端口转发配置保存为可复用模板

### 服务器监控

- **服务器仪表盘**：概览所有主机的连接状态
- **资源监控**：实时 CPU、内存等指标展示与历史快照

### AI 助手

- **AI 对话**：内置 AI 聊天面板，可解释终端输出、协助排查问题
- **预设配置**：可自定义 AI 预设与 API 密钥

### 界面与个性化

- **左侧面板**：会话列表、SFTP、传输队列、脚本管理、设置，标签页切换
- **自定义快捷键**：所有快捷键均可重新绑定，内置多套预设（默认、VS Code 风格、iTerm2 风格、无绑定）
- **主题系统**：统一的设计系统（颜色、间距、圆角、阴影）
- **多语言**：支持中文和英文
- **深色/浅色模式**：跟随系统或手动切换

### 安全

- **加密存储**：密码、私钥口令等敏感信息通过 FlutterSecureStorage 加密保存
- **主机密钥验证**：首次连接提示确认，密钥变更告警
- **SSH Agent**：支持系统 SSH Agent 认证

## 快速开始

### 下载

从 [Releases](https://github.com/PNOS-770/Asmote/releases) 下载对应平台的最新版本并安装。

### 从源码构建

**前置依赖**：Flutter SDK、Rust 工具链

```bash
git clone https://github.com/PNOS-770/Asmote.git
cd Asmote

# 编译原生核心
cd native/asmote_native_core
cargo build --release
cd ../..

# 运行
flutter run
```

### 平台构建

```bash
flutter build windows   # Windows
flutter build linux     # Linux
flutter build macos     # macOS
flutter build apk       # Android
flutter build ios       # iOS
```

## 使用指南

### 新建连接

1. 启动应用后，点击左侧面板顶部的 **+** 按钮
2. 选择连接类型：SSH / 本地终端 / 串口 / Telnet
3. 填写连接信息（主机地址、端口、认证方式等）
4. 点击连接

### 分屏操作

- **分割面板**：在面板标签栏右键或使用快捷键分割
- **切换面板**：`Ctrl+Alt+方向键` 切换焦点
- **最大化/还原**：`Ctrl+Alt+Enter`
- **广播输入**：`Ctrl+Alt+B` 切换广播模式
- **关闭面板**：标签栏右键关闭

### 文件传输

1. 在左侧面板切换到 **SFTP** 标签页
2. 连接到目标主机后即可浏览远程文件系统
3. 右键文件或文件夹可选择上传/下载
4. 也可拖拽文件到终端窗口上传

### 脚本管理

1. 在左侧面板切换到 **脚本** 标签页
2. 点击 **新建脚本** 创建脚本内容
3. 可在脚本中使用 `{{变量名}}` 作为参数
4. 选择目标主机执行，或配置定时调度/触发器自动执行

### 端口转发

1. 在设置中进入 **端口转发** 选项卡
2. 点击 **新增** 创建转发规则（本地/远程/SOCKS）
3. 配置监听地址、端口和目标地址
4. 可在主机编辑中设置自动启动的转发

### 自定义快捷键

1. 打开 **设置 → 快捷键**
2. 选择要修改的操作
3. 按下新的快捷键组合
4. 支持导入预设（VS Code 风格、iTerm2 风格等）

### AI 助手

1. 在设置中配置 AI 预设与 API 密钥
2. 点击工具栏或菜单中的 AI 图标打开 AI 面板
3. 可选中终端输出发送给 AI 进行分析

## 默认快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` / `Ctrl+C` | 复制 |
| `Ctrl+V` / `Shift+Insert` | 粘贴 |
| `Ctrl+A` | 全选 |
| `Ctrl+F` | 查找 |
| `Alt+B` | 切换块选择 |
| `Ctrl+D` | 分屏 |
| `Ctrl+Alt+Enter` | 最大化 / 还原窗格 |
| `Ctrl+Alt+B` | 切换输入广播 |
| `Ctrl+Alt+←/↑` | 上一个窗格 |
| `Ctrl+Alt+→/↓` | 下一个窗格 |

所有快捷键均可在 **设置 → 快捷键** 中自定义。

## 配置

配置文件存储在应用支持目录下的 `asmote_state.json`。推荐通过 UI 设置修改。

- **通用**：语言、布局、自动重连
- **终端**：字体/字号/行高、滚动缓冲区、终响铃、背景图片/不透明度
- **快捷键**：自定义所有键盘快捷键
- **端口转发**：管理 SSH 隧道模板
- **AI**：配置 AI 预设与 API 密钥
